## What is Open LLM VTuber?

**Open LLM VTuber** is a unique voice-interactive AI companion that combines real-time voice conversations, visual perception, and a lively Live2D avatar — all running completely offline on your computer.

You can treat it as your personal AI companion: a virtual character with a customizable appearance, voice, and persona. The project fully supports **Windows**, **macOS**, and **Linux**, and offers both a web browser version and a desktop client (with special support for **transparent background desktop pet mode**).

<Note>
  Open LLM VTuber is in active development. v2.0 is currently in early planning phase. The current stable version is v1.x.
</Note>

## Key features

<CardGroup cols={2}>
  <Card title="Fully offline" icon="shield-check">
    Run completely offline using local models. No internet required. Your conversations stay on your device.
  </Card>
  <Card title="Live2D avatar" icon="face-smile">
    A lively Live2D character with expression control driven by LLM output. Import your own custom models.
  </Card>
  <Card title="Voice interruption" icon="microphone">
    Hands-free voice interaction without headphones. AI won't hear its own voice — interrupt any time naturally.
  </Card>
  <Card title="Cross-platform" icon="desktop">
    Perfect compatibility with macOS, Linux, and Windows. Supports NVIDIA and non-NVIDIA GPUs, CPU, and cloud APIs.
  </Card>
  <Card title="Modular backends" icon="puzzle-piece">
    Swap LLM, ASR, and TTS engines via config file — no code changes needed. Dozens of backends supported.
  </Card>
  <Card title="Visual perception" icon="eye">
    AI can see your screen, camera feed, or screenshots and respond to what it observes.
  </Card>
  <Card title="Desktop pet mode" icon="paw">
    Transparent background, global top-most, and click-through. Drag your AI companion anywhere on screen.
  </Card>
  <Card title="MCP tools" icon="wrench">
    Model Context Protocol integration lets the AI use external tools like web search, time, and more.
  </Card>
</CardGroup>

## Supported backends

### Large Language Models (LLM)

Ollama, OpenAI, OpenAI-compatible APIs, Claude (Anthropic), Gemini, Mistral, DeepSeek, Groq, ZhiPu AI, LM Studio, llama.cpp (GGUF), vLLM, Letta, Hume AI EVI, and more.

### Automatic Speech Recognition (ASR)

Sherpa-ONNX, FunASR, Faster-Whisper, Whisper.cpp, OpenAI Whisper, Groq Whisper, Azure ASR.

### Text-to-Speech (TTS)

Edge TTS, Sherpa-ONNX TTS, pyttsx3, MeloTTS, Coqui-TTS, GPT-SoVITS, Bark, CosyVoice, CosyVoice2, Piper TTS, Fish Audio, Azure TTS, ElevenLabs, Cartesia, Minimax TTS, SiliconFlow, Spark TTS, OpenAI TTS.
